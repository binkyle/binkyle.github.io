## 部署到gitpages

若参考上面项目构建链接，此时本地git仓库已初始化

1.创建本地分支

  查看本地分支
```shell
git branch
```
  创建本地分支
```shell
git checkout -b master
```

2.连接远程分支

添加远程仓库的引用
```sh
git remote add origin https://github.com/yourusername/myrepo.git
```

3.提交本地分支到远程分支
```sh
git push origin <local_branch>:<remote_branch>
```
