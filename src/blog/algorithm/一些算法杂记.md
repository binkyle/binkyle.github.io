## 余法运算公式
a mod b表示a除以b的余数。有下面的公式：
- (a + b) % p = (a%p + b%p) %p
- (a - b) % p = ((a%p - b%p) + p) %p
- (a * b) % p = (a%p)*(b%p) %p
